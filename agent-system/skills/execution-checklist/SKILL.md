# Execution Checklist

## Objective

Prevent misses before release or handoff.

## Checklist

- Scope complete vs acceptance criteria
- Tests green and critical paths covered
- Security checks reviewed
- Docs and changelog updated
- Rollback path defined

## Output

- Pass/fail per checklist item
- Blocking issues
- Go/no-go recommendation

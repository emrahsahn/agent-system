# Decision Matrix

## Objective

Compare options objectively and choose with clear rationale.

## Steps

1. Define decision criteria (impact, effort, risk, time).
2. Score each option against each criterion.
3. Document trade-offs and second-order effects.
4. Recommend one option and fallback plan.

## Output

- Options list
- Criteria and weights
- Scoring summary
- Recommended decision and fallback
